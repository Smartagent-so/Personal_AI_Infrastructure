# BrandBuilder Skill - Konzept

**Version:** 1.0
**Datum:** 2025-12-03
**Basiert auf:** Gemini Brand Guide Research + CIS Integration

---

## 1. Executive Summary

BrandBuilder ist ein interaktives, CIS-gesteuertes Skill zur Entwicklung vollständiger Brand Guides. Anders als statische Frameworks nutzt BrandBuilder eine **adaptive Interview-Methodik** mit den 5 CIS-Agenten, die je nach Situation und Fortschritt des Users aktiviert werden.

**Kernprinzipien:**
- **Status-Quo Assessment zuerst** - Was weiß der User bereits?
- **Adaptive Methodenwahl** - Basierend auf vorhandenem Material
- **Milestone-Dokumente** - Jede Phase produziert persistierte Outputs
- **CIS-Agent-Interviews** - Echte Gespräche mit spezialisierten Personas

## 3. Die 5 Phasen (aus Gemini Research)

### Phase 1: Discovery (2-3 Wochen)
**Deliverables:**
- Discovery Document (80+ Fragen beantwortet)
- Competitor Analysis (5-10 Wettbewerber)
- Customer Personas (2-3)
- SWOT-Analyse

**CIS-Agents:**
- Dr. Quinn: "Warum existiert eure Marke?" (Five Whys)
- Maya: Empathy Interviews für Personas

### Phase 2: Strategy (2-3 Wochen)
**Deliverables:**
- Mission, Vision, Values
- Positioning Statement
- Brand Archetype
- Tone of Voice Guidelines
- Messaging Framework

**CIS-Agents:**
- Victor: Blue Ocean Strategy, Positioning Canvas
- Carson: Tagline Brainstorming
- Sophia: Brand Story Development

### Phase 3: Visual Identity (3-4 Wochen)
**Deliverables:**
- Logo System (Primary, Secondary, Monochrome)
- Farbpalette (Codes für alle Formate)
- Typografie-System
- Imagery Guidelines

**CIS-Agents:**
- Carson: Visual Brainstorming, Moodboard Development
- Maya: User Testing Feedback

### Phase 4: Documentation (1-2 Wochen)
**Deliverables:**
- Brand Guide Document (50-100 Seiten)
- Brand Asset Library
- Application Examples

**CIS-Agents:**
- Sophia: Narrative Sections (Brand Story, About)

### Phase 5: Implementation (2+ Wochen)
**Deliverables:**
- Business Cards, Email Signatures
- Social Media Templates
- Website Guidelines
- Team Training Materials

## 4. Adaptive Entry Points

Der User kommt mit unterschiedlichem Vorwissen. BrandBuilder analysiert den Status-Quo und empfiehlt den optimalen Einstieg:

### Szenario A: Komplett Neu
```
User: "Ich möchte eine neue Marke aufbauen"
→ Start bei Phase 1.1 (Brand Purpose Discovery mit Dr. Quinn)
```

### Szenario B: Bestehendes Business, kein Branding
```
User: "Ich habe mein Business seit 2 Jahren, aber nie formell gebrandet"
→ Schnelle Discovery (was existiert implizit?)
→ Start bei Phase 1.3 (Konkurrenzanalyse)
```

### Szenario C: Rebrand
```
User: "Wir wollen unsere Marke aktualisieren"
→ Brand Audit (was funktioniert, was nicht?)
→ Gezielter Einstieg in spezifische Phasen
```

### Szenario D: Spezifische Hilfe
```
User: "Ich brauche nur einen Tagline"
→ Kurzes Positioning-Check
→ Direkt Carson für Tagline-Session
```

## 5. Interview-Flow mit CIS-Agenten

### 5.1 Dr. Quinn Session: Brand Purpose Discovery

**Trigger:** Phase 1, wenn User keine klare Mission hat

**Flow:**
1. Quinn begrüßt, erklärt Five Whys Methode
2. Stellt erste "Warum?"-Frage
3. Iteriert 5x, gräbt tiefer
4. Findet Root Cause / Core Purpose
5. Formuliert Mission Statement Draft

**Beispiel-Dialog:**
```
Quinn: "Willkommen. Ich bin Dr. Quinn, und heute finden wir heraus,
       WARUM deine Marke existiert. Erzähl mir: Was macht dein Business?"

User: "Ich mache Website-Designs für kleine Unternehmen."

Quinn: "Interessant. Und WARUM machst du Website-Designs?"

User: "Weil kleine Unternehmen oft schlechte Websites haben."

Quinn: "Warum ist das ein Problem?"

User: "Weil sie dadurch weniger Kunden bekommen."

Quinn: "WARUM bekommen sie weniger Kunden durch schlechte Websites?"

User: "Weil die Website nicht professionell wirkt und Vertrauen fehlt."

Quinn: "AHA! Da haben wir es! Dein WHY ist nicht 'Websites bauen' -
       es ist 'Vertrauen schaffen für kleine Unternehmen'.
       Das ist dein Core Purpose."
```

### 5.2 Maya Session: Customer Empathy

**Trigger:** Phase 1, für Persona Development

**Flow:**
1. Maya führt empathisch durch Kundenverständnis
2. Jobs-to-be-Done Exploration
3. Pain/Gain Mapping
4. Persona-Erstellung

**Beispiel-Dialog:**
```
Maya: "Stell dir deinen idealen Kunden vor. Erzähl mir von einem echten Menschen,
       den du kennst, der perfekt zu deiner Zielgruppe passt."

User: "Maria, 45, hat eine kleine Bäckerei..."

Maya: "Was FÜHLT Maria, wenn sie über ihre Website nachdenkt?"

User: "Frustration. Sie weiß, dass sie eine braucht, aber..."

Maya: "Was ist der tiefere Schmerz dahinter? Was liegt ihr nachts wach?"
```

### 5.3 Victor Session: Strategic Positioning

**Trigger:** Phase 2, für Differenzierung

**Flow:**
1. Blue Ocean Strategy Canvas
2. Competitive Landscape Mapping
3. Value Proposition Canvas
4. Positioning Statement Development

**Beispiel-Dialog:**
```
Victor: "Zeig mir die Landschaft. Wer sind deine 5 wichtigsten Konkurrenten?"

User: [listet Konkurrenten]

Victor: "Interessant. Auf einer Skala von 1-10, wie würdest du diese
        Faktoren für jeden bewerten: Preis, Qualität, Speed, Service?"

Victor: "Ich sehe eine Lücke. Niemand bedient [X]. Das könnte dein Blue Ocean sein.
        Aber die entscheidende Frage: Kannst du das WIRKLICH besser als alle anderen?"
```

### 5.4 Carson Session: Creative Identity

**Trigger:** Phase 2-3, für Naming/Taglines/Visuals

**Flow:**
1. SCAMPER-geführtes Brainstorming
2. Divergentes Denken fördern
3. Konvergieren auf Top-Ideen
4. Refinement

**Beispiel-Dialog:**
```
Carson: "YES! Lass uns kreativ werden! Ich werfe erstmal 20 verrückte Ideen
        an die Wand. Keine Zensur! Bereit?"

Carson: "Name-Idee 1: 'TrustBuild'... 2: 'PixelTrust'... 3: Lass uns
        SCAMPER anwenden - was, wenn wir 'Website' mit 'Vertrauen' KOMBINIEREN?"

Carson: "Das ist BRILLIANT! Bau darauf auf! Was noch?"
```

### 5.5 Sophia Session: Brand Story

**Trigger:** Phase 2, 4 für Narrativ

**Flow:**
1. Hero's Journey Framework anwenden
2. User als Guide, Kunde als Held positionieren
3. Transformation Story entwickeln
4. Emotionale Beats einbauen

**Beispiel-Dialog:**
```
Sophia: "Jede große Marke erzählt eine Geschichte. Und in dieser Geschichte
        ist dein Kunde der HELD, nicht du. Du bist der weise Guide,
        wie Gandalf oder Yoda. Erzähl mir: Welche Transformation erlebt
        dein Kunde durch dich?"

Sophia: "Wunderschön. Ich höre die Geschichte von Maria, die frustriert war,
        keine Kunden fand... dann DU kamst... und jetzt? Wie endet ihre Geschichte?"
```

- [ ] 60+ Discovery-Fragen beantwortet
